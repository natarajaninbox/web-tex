Question 1

A signal given by x(t) = a1 sin(ωt + ϕ1) is applied to a stable linear time-invariant system. Suppose the steady state output response of the system is given by a2F(ω2t + ϕ2). Then which of the following is true about the function F? (2 marks)

  • F is not necessarily ’sine’ or ’cosine’, but must be periodic with ω1 = ω2.
  • F must be a ’sine’ or a ’cosine’ function with a1 = a2
  • F must be a ’sine’ function with ω1 = ω2 and ϕ1 = ϕ2
  • F must be a ’sine’ or ’cosine’ function with ω1 = ω2.

Question 2

A continuous-time system is described by y(t) = e−|x(t)|, where y(t) is the output and x(t) is the input. The output signal y(t) is bounded (2 marks)

  • only when x(t) is bounded.
  • only when x(t) is non-negative
  • only for t ≤ 0 if x(t) is bounded for t ≥ 0
  • even when x(t) is non-negative.
